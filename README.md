These Days
==========

for help creating periodic time capsules.

## Requirements
* `py2app`
* `rumps`

## How to run

Option 1. Run without installing
 * Run `python TheseDays.py`


Option 2. Install
 * Run `python setup.py py2app`
 * Now drag `dist/TheseDays.app` to your Applications folder--or just open it!
