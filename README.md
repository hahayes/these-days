these-days
==========

for remembering to create periodic time capsules

## requirements
* `py2app`
* `rumps`

## how to run

#### Option 1: run, don't install
* Run `python TheseDays.py`

#### Option 2: install
* Run `python setup.py py2app`
* Now drag `dist/TheseDays.app` to your Applications folder--or just open it!
